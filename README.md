This package provides a wonderful CDN helper to integrate any Laravel application with a CDN provider like Cloudflare.

> **Requires [PHP 8.3+](https://php.net/releases/)**

⚡️ Install package using [Composer](https://getcomposer.org):

```bash
composer require perseid/laravel-cdn
```

⚗️ Run static analysis using **PHPStan**:
```bash
composer test:types
```

✅ Run unit tests using **PEST**
```bash
composer test:unit
```

🚀 Run the entire test suite:
```bash
composer test
```
